### Installation

- First, clone this project
- Then run these commands: 
  - ``composer install``
  - ``npm install``
  - ``npm run dev``
- Configure .env
- Run migration
- Import data from ``~/database/seeders/fokontany_database_sql/`` then run these commands:
  - ``php artisan serve`` to launch your local server
  - ``npm run watch``
